# Info connectée - Le Journal Numérique

Bienvenue sur le dépôt du projet **Info connectée**, le journal numérique de l'école La Clairefontaine (CM1 B, 2025-2026). 

Ce site est conçu pour être élégant, moderne et facile à mettre à jour par les élèves et les enseignants.

## 🚀 Technologies utilisées
- **Hugo** : Générateur de site statique ultra-rapide.
- **HTML5 / Vanilla CSS** : Structure et design sur mesure (inspiré du style "Le Monde").
- **JavaScript** : Pour l'interactivité (menu burger, etc.).

## 📁 Structure du Projet
- `content/posts/` : C'est ici que se trouvent tous les articles du journal.
- `assets/css/style.css` : Le fichier de style qui définit l'apparence du site.
- `layouts/` : Les modèles HTML du site.
- `static/images/` : Dossier pour stocker toutes les photos des articles.

## ✍️ Comment créer un article ?
Un guide détaillé a été créé spécialement pour les contributeurs. Consultez le fichier :
👉 [**MODÈLE_ARTICLE.md**](MODÈLE_ARTICLE.md)

Il contient un modèle à copier-coller et toutes les règles à suivre (rubriques, images, etc.).

## 🖥️ Développement Local
Si vous souhaitez voir les modifications en temps réel sur votre ordinateur :
1. Installez [Hugo](https://gohugo.io/).
2. Lancez la commande suivante à la racine du projet :
   ```bash
   hugo server -D
   ```
3. Ouvrez votre navigateur sur `http://localhost:1313/`.
